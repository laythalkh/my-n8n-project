# n8n Workflow Project

This project contains an n8n automation workflow.

## Features
- Automated workflow built with n8n
- Easy to import and test
- Can be connected with external services

## Files
- workflow.json: the exported n8n workflow
- README.md: project description

## How to use
1. Open n8n
2. Import the workflow.json file
3. Configure your credentials
4. Run the workflow

## Notes
Do not forget to add your own API keys and credentials before running the workflow.
